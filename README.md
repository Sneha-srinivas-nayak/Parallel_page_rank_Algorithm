# Parallel_page_rank_Algorithm

Parallel implementation of Pagerank algorithm

# Requirements

Install Go 
Install Python 

# Installation

Clone the repository:
git clone https://github.com/Sneha-srinivas-nayak/Parallel_page_rank_Algorithm
Install any required dependencies:
go get ./...

# Usage
To run the project, use the following command:

go run . <num_of_workers>
This will compile and run the project. The output will be printed to the console.
